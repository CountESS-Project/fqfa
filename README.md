# fqfa

A lightweight Python library for handling [FASTQ](https://www.ncbi.nlm.nih.gov/sra/docs/submitformats/#fastq-files) and [FASTA]( https://www.ncbi.nlm.nih.gov/BLAST/fasta.shtml) files.
